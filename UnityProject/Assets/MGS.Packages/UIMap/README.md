# MGS.UIMap

## Summary
- Unity plugin for make mini map and draw motion trail in scene UI.

## Environment
- Unity 5.0 or above.
- .Net Framework 3.5 or above.

## Platform

- Windows.

## Version

- 1.1.0

## Achieve
- Global map UI.
- Local map UI.
- Flag on map.
- Draw trail of dynamic flag on map.

## Demo
- Demos in the path "MGS.Packages/UIMap/Demo" provide reference to you.
